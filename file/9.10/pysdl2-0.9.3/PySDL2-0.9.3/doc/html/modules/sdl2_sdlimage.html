sdl2.sdlimage - SDL2_image library wrapperÂ¶

The sdl2.sdlimage module is a ctypes-based wrapper around the SDL2_image library. It wraps nearly all publicly accessible structures and functions of the SDL2_image library to be accessible from Python code.

A detailed documentation about the behaviour of the different functions can found within the SDL2_image documentation.
