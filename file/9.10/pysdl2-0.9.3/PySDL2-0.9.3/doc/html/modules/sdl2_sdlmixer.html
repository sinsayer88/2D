sd2.sdlmixer - SDL2_mixer library wrapperÂ¶

The sdl2.sdlmixer module is a ctypes-based wrapper around the SDL2_mixer library. It wraps nearly all publicly accessible structures and functions of the SDL2_mixer library to be accessible from Python code.

A detailed documentation about the behaviour of the different functions can found within the SDL2_mixer documentation.
